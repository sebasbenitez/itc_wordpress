== Description ==

Este plugin permite a tus clientes autocompletar el 90% los datos del checkout obteniendo la informacion desde la AFIP / ARCA.

== Installation ==

1) Instala y activa el plugin en tu tienda.

 
Una vez instalado tus clientes podran ingresar su DNI / CUIT / CUIL en el checkout y automaticamente se completaran los datos con la info que proviene desde la AFIP / ARCA.
